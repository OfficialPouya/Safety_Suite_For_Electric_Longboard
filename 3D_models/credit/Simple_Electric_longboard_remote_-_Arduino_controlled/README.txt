# Summary

This is my attempt to make a simple and reliable remote for electric skateboards. The main electronics are an Arduino Nano, a Hall Effect sensor, and an nRF24 radio module. You can find a full parts list here: https://goo.gl/EXNYyZ

The software, documentation and schematics can be found on my github page: https://github.com/SolidGeek/nRF24-Esk8-Remote

I recommend printing the bottom and top shell top-down, so you don't need any supports - and gets a nice finish if you are printing on glass. 

Hope you enjoy! 

EDIT: A lot have been asking for the source-files to make modifications, they can be downloaded here (inventor-files): https://drive.google.com/drive/folders/1Mf0LRRubAH5jnB08VI2jshMJtrSSzVIn
